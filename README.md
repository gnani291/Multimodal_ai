AI Lecture Notes Generator
🚀 Overview

AI Lecture Notes Generator is an AI-powered multimedia application that automatically converts lecture videos, audio recordings, and educational content into structured notes and summaries.

The project integrates Speech Recognition, Natural Language Processing, OCR, and Text-to-Speech technologies to simplify learning and content understanding.

Built using Python and Streamlit, the application demonstrates real-world implementation of Generative AI and multimedia processing.

✨ Features
🎥 YouTube Video Downloader

Download lecture videos directly from YouTube using:

yt-dlp
🎤 Speech-to-Text Conversion

Convert lecture audio into text using:

Whisper
📝 AI-Powered Notes & Summarization

Generate concise lecture notes and summaries using:

Transformers
🖼️ OCR Text Extraction

Extract text from lecture slides or images using:

Tesseract OCR
🔊 Text-to-Speech

Convert generated notes into audio format using:

gTTS
